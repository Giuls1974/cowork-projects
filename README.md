## up2you-video

Video pubblicitario B2B per Up2YOU – Bilancio di Sostenibilità ESG.

**Stack:** Python · Pillow · FFmpeg
**Output:** 15s · 1920×1080 · H.264 · YouTube-ready

### Render

```bash
python3 render.py
```

Richiede: `Pillow`, `ffmpeg`, font Poppins e FontAwesome in `/usr/share/fonts/`.
